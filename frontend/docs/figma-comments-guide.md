# Инструкция: Добавление Комментариев в Figma

Откройте Figma файл: https://www.figma.com/design/wQrx3oBYscQFtyvPhRj5Mu?node-id=3-2

## Как Добавить Комментарий

1. Выберите фрейм/элемент в Figma
2. Нажмите `C` или кликните иконку комментария
3. Вставьте текст из списка ниже
4. Нажмите "Post"

---

- Кликнуть на элемент в Figma
- Увидеть комментарий с путем к файлу
- Открыть файл и найти нужный код
